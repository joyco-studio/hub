## About this Component

The **Media Player** component from [Dice UI](https://www.diceui.com) provides a fully-featured media playback experience with:

- Video and audio playback support
- Customizable controls and styling
- Built-in accessibility features
- Responsive design

## Installation

Visit the [Dice UI Media Player documentation](https://www.diceui.com/docs/components/media-player) for installation instructions and usage examples.

```bash
# Install via the Dice UI registry
npx shadcn@latest add https://www.diceui.com/r/media-player.json
```

## Why We Recommend This

We love this component because it handles all the nuances of media playback that are easy to get wrong - keyboard navigation, screen reader support, and consistent behavior across browsers.

Rather than reinventing the wheel, we're pointing you to the excellent work done by the Dice UI team.
